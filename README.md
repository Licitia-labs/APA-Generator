# APA-Generator
De utilidad para todo investigador y resolver la manera de hacer citas con APA de diversas fuentes primarias y secundarias. 
La investigación cientifica es importante. Este modelo resuelve las dudas para el postulante.
En un archivo Word, tiene una limitante y es el APPA 6. ESte modelo esta actualizado al APA7
De facil acceso. La acumulación de citas, se trasladan a un archivo Word.
